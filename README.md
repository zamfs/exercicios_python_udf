# 🐍 Exercícios de Python — Lógica de Programação

Repositório com listas de exercícios desenvolvidas durante meus estudos de lógica de programação utilizando Python.

O foco principal deste projeto foi praticar a construção da lógica manualmente, evitando o uso excessivo de funções prontas ou soluções muito simplificadas.  
A ideia era fortalecer o raciocínio lógico e entender como os algoritmos funcionam na prática.

---

## 📚 Conteúdo das listas

### 📌 Lista 1 — Conceitos Básicos
Exercícios introdutórios sem utilização de estruturas condicionais (`if`, `else`) ou laços.

- Entrada e saída de dados
- Operações matemáticas
- Variáveis
- Conversões de tipos

---

### 📌 Lista 2 — Estruturas Condicionais
Exercícios utilizando:

- `if`
- `else`
- `elif`

Foco em tomada de decisão e validações.

---

### 📌 Lista 3 — Estruturas de Repetição (`while`)
Exercícios envolvendo:

- Laços `while`
- Contadores
- Acumuladores
- Validação de entrada
- Sequências numéricas

---

### 📌 Lista 4 — Estruturas de Repetição (`for`)
Exercícios utilizando:

- Laços `for`
- Iterações
- Tabuadas
- Séries numéricas
- Padrões e repetições

---

### 📌 Lista 5 — Funções (`def`)
Exercícios focados em:

- Criação de funções
- Parâmetros
- Retorno de valores
- Organização e reutilização de código

---

## 🎯 Objetivo

Desenvolver a base da lógica de programação antes de utilizar recursos mais avançados da linguagem.

Por isso, muitos exercícios foram resolvidos de forma mais “manual”, priorizando o aprendizado da lógica ao invés da solução mais curta possível.
